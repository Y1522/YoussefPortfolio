# Youssef's 3D Portfolio

A modern, interactive portfolio website built with React and Three.js featuring smooth animations and 3D elements.

## Features

- Responsive design for all device sizes
- Interactive 3D elements using Three.js
- Smooth page transitions and animations with Framer Motion
- Glass-like UI elements
- Four main sections: Home, About, Resume, and Contact

## Technologies Used

- React.js
- Three.js / React Three Fiber / Drei
- Styled Components
- Framer Motion for animations
- React Intersection Observer

## Setup and Installation

1. Clone the repository
2. Install dependencies:
```
npm install
```
3. Start the development server:
```
npm start
```
4. Build for production:
```
npm run build
```

## Project Structure

- `src/components/` - React components
  - `Header.js` - Glass-effect navigation header
  - `Home.js` - Landing section
  - `About.js` - About me section
  - `Resume.js` - Resume/experience section
  - `Contact.js` - Contact form section
  - `FloatingModel.js` - 3D model component

## Customization

To customize this portfolio:

1. Update personal information in each component
2. Replace the placeholder images
3. Modify the color scheme in `index.css`
4. Add or remove sections as needed
