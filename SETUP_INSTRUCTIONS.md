# Setup Instructions

To complete the setup of your portfolio website, follow these steps:

## 1. Copy Your Files

Copy the following files to the specified locations:

1. Copy your CV file "Youssef_CV_final.pdf" to:
   `src/assets/Youssef_CV_final.pdf`

2. Copy your profile photo "my foto ( cv).png" to:
   `src/assets/my foto ( cv).png`

## 2. Run the Application

After copying the files, run the following commands in the project directory:

```bash
npm install
npm start
```

This will start the development server and open the website in your browser.

## Features Added

- The "Let's Talk" button in the Home section now scrolls to the Contact section
- The "Download Full CV" button in the Resume section now downloads your CV
- Your profile photo is displayed in the Home section
- Smooth scrolling behavior is enabled for a better user experience

## Customization

You can further customize the website by editing the component files in the `src/components` directory. Each component represents a section of your portfolio.
